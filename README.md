# Clove Collective

Official marketing website for Clove Collective, a Philippine PR and talent representation agency.

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production

```bash
npm run build
npm start
```

## Project structure

- `app/` — website pages, layout, and styling
- `public/` — brand assets and campaign media
- `config/` — deployment configuration
- `build/` — production build integration
